## üìã Resume Sections

### Section 1: Personal Information
- **Full Name** (Required)
- Email, Phone, Address
- Country, Website, LinkedIn
- Professional Headline
- Professional Summary (2-3 sentences)

### Section 2: Education
- Institution Name
- Degree type (Bachelor, Master, PhD, etc.)
- Field of Study
- Start/End dates
- GPA/Grade (optional)
- Description (achievements, honors)

### Section 3: Work Experience
- Company Name
- Job Position/Title
- Employment Dates
- Job Description
- Key Achievements (bullet points)

### Section 4: Skills
**Technical Skills**
- Programming languages (Python, JavaScript, etc.)
- Tools and frameworks
- Software expertise

**Soft Skills**
- Communication, Leadership
- Problem Solving, Teamwork
- Project Management

### Section 5: Languages
- Language Name
- **CEFR Proficiency Level** (A1-C2)
  - **A1-A2**: Beginner to Elementary
  - **B1-B2**: Intermediate to Upper Intermediate
  - **C1-C2**: Advanced to Proficient
- Certification (TOEFL, IELTS, DALF)
- Certification Date

### Section 6: Certifications & Projects
**Certifications**
- Name, Issuing Organization
- Issue/Expiration Dates
- Credential ID, URL

**Projects**
- Project Name, Description
- Technologies Used
- Project URL/GitHub Link

## ‚ú® Pro Tips

### Best Practices

1. **Personal Info**
   - Use a professional email address
   - Include LinkedIn profile
   - Write a compelling headline

2. **Education**
   - List most recent first
   - Include GPA if it's 3.5 or higher
   - Add relevant coursework or honors

3. **Work Experience**
   - Use action verbs: Led, Developed, Increased, Managed
   - Add quantifiable metrics: "Increased sales by 35%"
   - Highlight achievements, not just tasks

4. **Skills**
   - Tailor to job descriptions you're applying for
   - Start with most relevant skills
   - Add 5-10 technical + 5-10 soft skills minimum

5. **Languages**
   - Include native language
   - Add certifications for credibility
   - European employers value multilingual abilities

6. **Certifications**
   - Include relevant professional credentials
   - Add online courses (Coursera, edX, etc.)
   - Include expiration dates if applicable

### Keyboard Shortcuts
- **Tab**: Move to next field
- **Shift+Tab**: Move to previous field
- **Enter** (in suggestions): Add skill/language
- **Click section**: Expand/collapse form

## üîí Data & Privacy

### Data Storage
- **Location**: Your browser's local storage
- **Access**: Only you can access your data
- **Privacy**: No data sent to servers (unless you enable optional features)
- **Backup**: Export as PDF to keep copies

### Clearing Data
- Data persists until you delete it
- Delete individual resumes from the home page
- Clear all resumes by clearing browser storage
- Export PDF before clearing important resumes

## üêõ Troubleshooting

### Resume won't save
- Check internet connection
- Try manual save button
- Check browser storage quota
- Try different browser if problem persists

### Preview not updating
- Check form is selected/expanded
- Make sure autosave is enabled
- Refresh page (F5)
- Check browser console for errors

### PDF download not working
- Check pop-up blocker settings
- Try different browser
- Ensure JavaScript is enabled
- Check available disk space

### Form section not responding
- Scroll to ensure field is visible
- Click on section title to expand
- Refresh page
- Try different browser

### Data disappears
- Check if using private/incognito mode
- Private mode data is cleared when closed
- Use normal browsing mode
- Export PDF as backup
